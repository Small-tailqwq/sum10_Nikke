# 1. 安装依赖
pip install fastapi uvicorn websockets

# 2. 启动大脑
python god_brain.py
*你会看到显示 `Sum10 外置大脑启动中...`，这表示它已经在后台待命了。*

**2. 启动控制台 (前端)**
直接在浏览器中打开生成的 `index.html`。

**新功能说明：**
* **☢️ 线程滑块 (Threads)**：这是最“猛”的设置。如果你有 8 核 CPU，拉到 8 或更高。后端会同时开启 8 个并行的求解进程，每个都在以不同的随机种子进行狂暴遍历（Race Mode）。
* **实时同步 (Real-time Sync)**：你不用等它们全跑完。只要任何一个线程发现了一个更高的消除率，WebUI 上的“Best Rate”会瞬间更新。
* **内存保护**：所有的计算压力都在 Python 进程中，浏览器只负责显示，再也不会爆内存了！

去吧，让 CPU 满载运行，榨干每一个数字！🔥
